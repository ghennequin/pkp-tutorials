val angles : Owl.Mat.mat

Neurons are "positioned" around a ring of orientation preference; angles is a vector whose ith element contains the preferred orientation of the ith neuron.

type radians = float
val rad_of_deg : float -> radians

Converts degrees into radians.

LGN input

val input_from_LGN : float -> radians -> Owl.Mat.mat

input_from_LGN contrast angle models LGN input as a vector of size n, when an oriented stimulus (angle) is presented at some contrast (a number typically between 0 and 1). Try it, plot it!

V1 model

val firing_rate_function : float -> float

Describes the input-to-rate transformation (at steady state) in each V1 neuron. In this model, the maximum firing rate is "1" by convention ─ biologically, "1" would actually mean ~80 Hz or so.

type prms = {
a : float;
b : float;
}

Network parameters; a and b describe recurrent interactions, such that the connection from neuron j to neuron i has strength -a + b cos(2*(angle(j) - angle(i))).

val feedforward_prms : prms

Parameters of the feedforward model (a=0, b=0, ie. no recurrent connectivity!).

val recurrent_prms : prms

Default parameters of the recurrent model.

val simulate : duration:float -> prms -> (float -> Owl.Mat.mat) -> Owl.Mat.mat * Owl.Mat.mat

simulate ~duration prms input simulates the network dynamics for duration seconds. Here, input is a function of time, which for any time t returns the LGN input to the network. Returns (time, rates), where time is a column vector of T time points, and rates is a matrix of size T x n.

Plotting functions

val angular_axis : Gp.property list

Set of plot properties you can use whenever you want to plot something as a function of angle.

val plot_dynamics : ?⁠movie_duration:float -> (Owl.Mat.mat * Owl.Mat.mat) -> unit

plot_dynamics ?movie_duration (time, rate) shows a movie of network activity as it unfolds over time, for a pair (time, rate) obtained from the simulate function.

val plot_steady_states : (Owl.Mat.mat * Owl.Mat.mat) list -> unit

plot_steady_state list plots multiple firing rate responses in the same graph. The argument is a list of (time, rate) pairs (e.g. each returned by the simulate function), and for each, only the final (steady-state) response is shown on the graph (instead of a movie).

Miscellaneous

val normalise : Owl.Mat.mat -> Owl.Mat.mat

Rescales a vector of firing rates so that the maximum value becomes one.
