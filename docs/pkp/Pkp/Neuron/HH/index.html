val simulate : prms:prms -> duration:float -> (float -> float) -> Owl.Mat.mat * Owl.Mat.mat

Simulates the model for a certain duration, with a certain input current (provided as a function of time in seconds).

returns

(time, state) where time is a column vector of times and state is a matrix containing the four state variables of the model at those same times (thus time and state have the same number of rows). The first column is the membrane potential (in mV), the other columns are the gate variables m, h, n in this order, optionally followed by the a and b gates in the case of the Connor-Stevens model.
