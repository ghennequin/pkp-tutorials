Module type Reinforcement_learning.Mission

Student: your MISSION is to implement each element of the following module:

val other_mark : mark -> mark

A function that, given a mark (X or O), returns the other one

val empty_board : board

An empty board -- cf board type above.

val mean : float list -> float

The mean of a list of numbers

val max_pair_list : ('a * float) list -> 'a * float

For a list of pairs (a1, b1); (a2, b2); ... , return the pair that has the largest b value. Hint: use List.fold_left

val transpose : board -> board

The (matrix) transpose of a board

val is_full : board -> bool

Check whether a board is full -- i.e. that here is no single None element in it.

val play_game : ?⁠display:(board -> unit) -> finished:(board -> bool) -> (player * player) -> board

play_game ?display (player1, player2) should implement a loop that alternates between player1 (to start) and player2, starting from empty_board; the function should return the final board configuration. If the optional argument display is provided, you should use it to display the current board at each iteration. You also have access (in the arguments) to finished, which says whether the current board corresponds to a terminated game.
